Meu Projeto Pizzaria La Gulla
====================

Sistema para a Pizzaria La Gulla onde será feito:
- Cadastro de Cliente
- Cadastro de Bairro
- Cadastro de Usuário
- Cadastro de Produto
- Movimentação: Abertura de Pedidos
- Movimentação: Fechamento de Pedidos
- Relatório: Cliente
- Relatório: Produto
- Relatório: Bairro
- Relatório: Usuários
- Relatório: Pedidos por Cliente
